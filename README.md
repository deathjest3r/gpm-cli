gpm-cli - Google Play Music Command Line Interface
==================================================

A command line interface to Google's Play Music service

Requirements
------------
* Python 2
* gmusicapi
* mplayer
* Some songs on Google Play Music

Usage
-----
./gpm-cli play "Slayer | Reborn"
